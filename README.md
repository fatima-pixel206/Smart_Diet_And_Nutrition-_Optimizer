 # 🥗 Smart Diet & Nutrition Optimizer (LP Based)
# 🚀 Overview
The Smart Diet Optimizer is an intelligent system designed using Linear Programming (LP) to suggest the ideal diet plan based on a user's budget and nutritional requirements. This project is specifically tailored for individuals aiming to achieve their health and fitness goals (Protein, Fats, Calories) while staying within a limited budget.
# ✨ Key Features
Budget-Friendly Optimization: Users can specify their price range, and the system calculates the most cost-effective food combinations.
Nutrient Tracking: Real-time calculation of Protein, Fats, and Calories.
Smart Filtering: Utilizes Java Collections and DSA algorithms for high-speed searching and filtering.
Interactive UI: A modern HTML5/CSS3 interface seamlessly connected via Spring Boot and Thymeleaf.
Mathematical Precision: Implements Linear Programming (Simplex Logic) to ensure mathematically optimized results.
# 🛠️ Tech Stack
